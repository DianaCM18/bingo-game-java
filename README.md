# Bingo Game (Java)

## Descripción
Aplicación de consola desarrollada en Java que simula un juego de bingo.

## Tecnologías utilizadas
- Java

## Funcionalidades
- Generación automática de cartón sin números repetidos
- Extracción aleatoria de números del 1 al 99 sin duplicados
- Comprobación de línea (5 aciertos) y bingo (10 aciertos)
- Interacción con el usuario por consola
